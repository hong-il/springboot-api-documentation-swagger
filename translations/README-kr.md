# 스프링부트 Swagger 튜토리얼
스웨거는 웹을 통한 API 테스트 뿐만 아니라,  
RESTful API 문서 자동화에도 용이합니다.

## 개발 환경  
개발환경은 다음을 참고해주세요.  
* IDE : IntelliJ IDEA Ultimate
* OS : Windows 10
* Spring Boot 2.2.2
* Java8
* Gradle
* Swagger2
## 목표  
* 스웨거의 기초 이해  
* 테스트와 문서화의 중요성 이해
